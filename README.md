# Features

ターゲット：女子中高生、20代女性<br>
YouTubeを見る理由ランキング<br>
<br>
No.1<br>
暇つぶしのため<br>
<br>
No.2<br>
情報収集。体験レビューなどリアルな声が聞けるから<br>
<br>
No.3<br>
ジャンル問わず楽しめる、簡単操作でラク<br>
<br>
<br>
解決ポイント
<br>
1つのアプリでラクラク操作<br>
・YouTubeの動画閲覧<br>
・情報取集ができる<br>
<br>
YouTuberファン同士(横の繋がり)で交流<br>
・ファンイベント前に友達を作れちゃいます！<br>
<br>
ジャンル毎のまとめ<br>
・だらだらしたい時！何も考えずに楽しめちゃう！<br>
